# Umgodi 🍺

**After hours, sorted.**

Umgodi is a late-night alcohol delivery platform for South Africa, starting in Nelspruit, Mpumalanga.

Built by Standford Elizabeth Group (Pty) Ltd.

## 🚀 Live
- [umgodi.co.za](https://umgodi.co.za)
- [umgodi.com](https://umgodi.com)

## 📱 Features
- Full SA alcohol catalogue (48 products)
- Zone-based delivery pricing (Zone 1: R50 | Zone 2: R65)
- Photo-verified deliveries
- SnapScan & PayFast payments
- Driver dashboard
- PWA — installable on iPhone & Android

## 🗺️ Delivery Zones (Nelspruit)
- **Zone 1** — CBD, Sonheuwel, Berghof (0–5km) — R50
- **Zone 2** — West Acres, Kamagugu, Valencia, Mataffin (6–12km) — R65
